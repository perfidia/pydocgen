The following styles are supported for Document:
	- page-numbering: bool
	- page-width: integer (mm)
	- page-height: integer (mm)
	- margin-top: integer (mm)
	- margin-bottom: integer (mm)
	- margin-left: integer (mm)
	- margin-right: integer (mm)
	- font-size: integer (pt)
	- font-name: string ("Times New Roman", "Arial", "Computer Modern", "DejaVu Serif")

The following styles are supported for Paragraph:
	- alignment: Alignment
	- text-indent: integer (pt)
	- margin-top: integer (pt)
	- margin-bottom: integer (pt)
	- margin-left: integer (pt)
	- margin-right: integer (pt)

The following styles are supported for Span:
	- font-name: string
	- font-size: integer (pt)
	- font-effects: FontEffect | ...
	- color: string (e.g. "#1234ff")
	- background-color: string (e.g. "#1234ff")

The following styles are supported for List:
	- list-style: ListStyle
	- item-spacing: integer (pt)
	- item-indent: integer (pt)
	- margin-top: integer (pt)
	- margin-bottom: integer (pt)
	- margin-left: integer (pt)
	- margin-right: integer (pt)
	- bullet-char: BulletChar or char
	
The following styles are supported for Image:
	- width: float (mm)
	- height: float (mm)
	- margin-top: integer (pt)
	- margin-bottom: integer (pt)
	- margin-left: integer (pt)
	- margin-right: integer (pt)
	- alignment: Alignment
	- seq-number-sep: string (e.g. "." -> "2.1.1", "> " -> "2> 1> 1")
	- caption-title: string or Span or list of Span (e.g. "Picture %d:\n")
